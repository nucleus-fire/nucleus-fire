# Nucleus Dashboard Example

A real-time analytics dashboard showcasing heavy data visualization and complex UI layouts.

## Features
- **Data Visualization**: CSS-only bar charts fueled by live data simulation.
- **Dark Mode**: Native dark-themed UI with slate/zinc palette.
- **Glassmorphism**: Translucent panels and overlays.
- **Simulated WebSocket**: Random data jitter to mimic live traffic updates.

## Tech Stack
- **Framework**: Nucleus V3
- **Styling**: Tailwind CSS (custom config)
- **Layout**: CSS Grid + Flexbox

## Running the Demo
```bash
# Run the server
nucleus run
```

Visit `http://localhost:3000`. Watch the graph bars animate in real-time as "traffic" hints the system.
